# Version 4 — Release & Closeout

**Release:** V4.0.0
**Status:** Complete.

> This document records what was shipped in Version 4. For instructions on resuming or extending the project, see [`architecture.md`](architecture.md) and [`roadmap.md`](roadmap.md).

## 1. What was built

Version 4 introduced the **FastAPI REST API**, exposing our existing services over HTTP. This milestone was the ultimate test of our architectural choices, proving that strict separation of concerns allows adding an entirely new presentation interface without modifying core logic.

### New Presentation Layer (`src/app/api/`)
- Added `fastapi` and `uvicorn` dependencies.
- Created `api/main.py` containing the FastAPI application, global error handlers for `ConfigError`, and a clean initialization sequence.
- Created `api/dependencies.py` to seamlessly bridge FastAPI's `Depends` system with our Custom DI `Container`.
- Established fully-documented routers (`crypto` and `football`) that call `CryptoService` and `FootballService`.

### Result Type Routing
- The API routers natively match against the `Result` objects (`Ok` and `Err`) generated by the service layer.
- `Err(APIError)` states are gracefully mapped into HTTP 502 Bad Gateway exceptions using `fastapi.HTTPException`.

### ADR-014: Native Dataclasses
- Instead of building duplicate Pydantic `BaseModel` schemas for API responses, we leveraged FastAPI's native support for standard library `dataclass`es. Our existing Domain models (`CryptoPrice`, `Tournament`) are directly used as `response_model` parameters, reducing boilerplate and maintaining a single source of truth.

### API Testing
- Built a comprehensive API test suite in `tests/api/test_routes.py` using `TestClient`.
- Tests run completely offline and fast by overriding FastAPI dependencies (`app.dependency_overrides`) with our test fakes (`FakeCryptoService`, `FakeFootballService`). 
- Test count is now 54 tests passing completely green on Python 3.12.

This lays the foundation for future clients, including a Web Dashboard (V5) or a Telegram Bot integration!
